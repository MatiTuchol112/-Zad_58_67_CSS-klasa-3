Pseudoelement: first-line

Obejmuje formatowaniem całą pierwszą linię
przykład:

p
 {
 color:navy;
 font-size:12pt;
 }

p:first-line
 {
 color:red;
 font-size:20pt;
 font-weight:bold;
 }

Pseudoelementy: after oraz :before.

Oba elementy służą do umieszczania tekstu przed i za formatowanym elementem strony
przykład:

p:before
 {
 content:"TO SĄ ĆWICZENIA";
 font-size:15pt;
 font-style:italic;
 font-weight:bold;
 }
p:after
 {
 content:"AUTOR Król Lew";
 font-size:15pt;
 font-style:italic;
 font-weight:bold;
 }

Pseudoelement: first-letter

Zastosowanie tego spowoduje sformowanie pierwszej litery w tekście
przykład:

p
 {
 color:navy;
 font-size:30pt;
 }
 p:first-letter
 {
 color:red;
 font-size:100pt;
 }
